# 二维码生成器

一个功能强大的二维码生成器应用，支持单个和批量生成高清二维码，提供打包下载功能，并支持特定格式文本的智能解析与排版。

## 🌟 功能特性

### ✅ 核心功能
- **高清生成**: 默认生成 800x800 分辨率的高清二维码，放大依然清晰锐利
- **单个生成**: 输入文本生成单个二维码
- **批量生成**: 支持多行文本批量生成多个二维码
- **批量下载**: 一键将多个二维码打包为 ZIP 文件下载，文件名自动包含序号和信息
- **实时预览**: 即时显示生成的二维码及排版效果

### ✅ 特殊格式智能解析
当你勾选了“特定格式解析”后，系统会对带有 `----` 的长文本进行智能截取和排版：
- **底部红字显示**: 自动提取 `----` 前的两段内容（如账号和密码），以醒目的红色大字显示在图片底部。
- **智能防裁切**: 底部文字会根据文本长度自动缩放字号，保证超长文字也能完整显示，绝不被裁剪。
- **纯净二维码**: 二维码本体的扫描内容，**只包含第三段及以后的文本**，过滤掉底部的账号和密码信息。
- **智能序号**: 生成的二维码图片左上角会自动打上 `#1`, `#2` 等序号标记。

### ✅ 用户体验
- **自动缓存**: 你的输入内容、批量文本和选项设置都会自动持久化到浏览器本地缓存中，刷新页面也不会丢失
- **纯净视图**: 每次点击生成新的二维码时，系统会自动清空下方旧的生成记录，保持页面清爽
- **模式切换**: 单个模式和批量模式之间自由切换
- **快捷键支持**: `Ctrl+Enter` 快速生成
- **文本复制**: 一键复制原始文本
- **响应式设计**: 适配各种屏幕尺寸

## 🚀 部署指南 (Vercel)

本项目是纯静态前端应用，基于 Vite 构建，**原生支持零配置部署到 Vercel**。

1. 将代码推送到 GitHub。
2. 登录 Vercel 后台，点击 "Add New Project"。
3. 导入本仓库。
4. Vercel 会自动识别出 Vite 框架：
   - **Build Command**: `pnpm run build` 或 `npm run build`
   - **Output Directory**: `dist`
5. 点击 **Deploy**，等待部署完成即可。

## 💻 使用方法

### 批量生成与打包下载
1. 点击"批量模式"按钮切换到批量模式
2. 勾选“特定格式解析”
3. 粘贴多行带 `----` 格式的文本
4. 点击"批量生成"按钮
5. 生成完毕后，点击界面上的 **"批量下载ZIP"** 即可一键获取所有图片

## 🛠️ 技术栈

- **Frontend**: React 18 + TypeScript + Vite 6
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **QR Code Generation**: qrcode
- **File Processing**: JSZip + file-saver
- **Notifications**: Sonner
- **Icons**: Lucide React
